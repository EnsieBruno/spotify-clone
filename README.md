# Spotify Clone 🎧

Este projeto é uma recriação da página inicial do Spotify utilizando HTML e CSS.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estruturação da página.
- **CSS3**: Estilização e layout.
